# IIUM Schedule Website

This website serves landing page for the IIUM Schedule app, documentation & guides and some tools. It is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

## Getting Started

```shell
pnpm install
```

Then,

```shell
pnpm run start
```
